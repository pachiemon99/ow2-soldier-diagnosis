## What changed

Every target/enemy detail diagnosis now has a deeper execution layer:

1. Read the matchup archetype.
2. Check the first condition before entering.
3. Commit only inside a short window.
4. Reset with a named escape or route rule.
5. Review one practical question after each fight.

This package also extends the diagnostic uniqueness guardrail so text does not remain as a short name-swapped template.
