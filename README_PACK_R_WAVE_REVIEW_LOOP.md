# OW Coach v50.18 Pack R: Wave Review Loop

Pack R adds a one-wave review loop to OW Coach.

## Added

- `owcoach_wave_review_loop_db_v50_18.csv`
- `owcoach_wave_review_loop_contract_v50_18.json`
- `WAVE_REVIEW_LOOP_CONTRACT_v50_18.md`
- `tests/owcoach-wave-review-loop-static.cjs`

## What changed

Each detail diagnosis now includes:

1. The goal for the next wave.
2. A review question.
3. A success signal.
4. A common mistake shape.
5. The next adjustment.
6. A replay/VOD marker.
7. A short micro drill.

Composition diagnosis also gets a compact one-wave review summary for the selected enemy five.
